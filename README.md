Project Prerequisites:
1. .NET 10 SDK installed
2. terminal (Command Prompt, PowerShell, or bash)
3. web browser

How to run and test

1. Clone repo
2. Type "dotnet run" in terminal and open the URL shown in the terminal output (e.g. http://localhost:5077)
3. Type any number and hit convert to get the text for that number


To run simple unit test

1. dotnet test NumToTextconverter.Tests/NumToTextconverter.Tests.csproj
2. 8 Test cases has been setup with multiple scenarios and edge cases to determine controller works as expected.
3. Preview test result in terminal console.
